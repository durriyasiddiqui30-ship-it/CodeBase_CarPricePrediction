Car Price Prediction using Machine Learning

Project Overview:
This project focuses on predicting car selling prices using Machine Learning techniques. A Linear Regression model was developed to estimate car prices based on vehicle characteristics such as manufacturing year, present price, kilometers driven, fuel type, seller type, transmission type, and ownership details.

Dataset Information:
The dataset contains 301 records and 9 variables.

Features:
•	Car_Name
•	Year
•	Present_Price
•	Driven_kms
•	Fuel_Type
•	Selling_type
•	Transmission
•	Owner
Target Variable
•	Selling_Price

Tools and Libraries Used:
•	Python
•	Pandas
•	NumPy
•	Matplotlib
•	Scikit-learn

Project Workflow:
1.	Data Loading
2.	Data Exploration
3.	Missing Value Analysis
4.	Data Preprocessing
5.	Label Encoding of Categorical Variables
6.	Train-Test Split
7.	Linear Regression Model Training
8.	Price Prediction
9.	Model Evaluation
10.	Visualization of Results

Model Performance
Evaluation Metrics:
•	Mean Absolute Error (MAE): 1.2147
•	Mean Squared Error (MSE): 3.5370
•	R² Score: 0.8465
The R² score indicates that approximately 84.65% of the variation in car selling prices is explained by the model.

Key Findings:
•	The dataset contained no missing values.
•	Vehicle characteristics significantly influence selling price.
•	The Linear Regression model achieved strong prediction accuracy.
•	Predicted prices closely matched actual selling prices.

Real-World Applications:
•	Car dealerships
•	Online automobile marketplaces
•	Vehicle valuation services
•	Insurance companies

Conclusion:
This project demonstrates the application of Machine Learning in automobile price prediction. The Linear Regression model achieved strong performance and can assist businesses and customers in estimating fair car prices based on vehicle attributes.
